Эта программа выполняет функцию игры "Виселица"
Описание игры: Игрок пытается угадать слово, предложенное компьютером, по буквам. За каждую ошибку игроку начисляется штраф, и он приближается к проигрышу. Программа создана без визуальной части с повешиванием, но присутствует интерфейс. В коде игры существует 20 слов для угадывания, которое выбирается случайным способом.
